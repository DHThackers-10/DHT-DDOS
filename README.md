# 🔧 How It Works

1. Launch tool
2. Input full URL (http:// or https://)
3. Tool resolves IP & starts attack instantly
4. Sends real traffic using sockets (UDP + HTTP)
5. Shows packet counter in real time
6. Ends when user presses Ctrl+C
